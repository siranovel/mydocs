DID(Differene in Difference)
============================
差分の差分法

## 概要

２時点以上の時点で観測されたデータを利用し、処理の前で「処理群の平均的な変化(差分)」と「対照群の平均的な変化(差分)」の差分を処理の効果として推定する手法です。  

## 使い方

```ruby
yi = [2261, 2458, 3904,2547]
ti = [0, 1, 0, 1]
zi = [0, 0, 1, 1]
regana = Num4RegAnaLib::RCTLib.new
regana.did(yi, ti, zi)
```

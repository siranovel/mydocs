oneway_anova
------------
一元配置分散分析

## 統計的検定
### 検定の手順1

・帰無仮設：水準A1, ..., Aa間の母平均に差が無い  
・対立仮説：水準A1, ..., Aa間の母平均に差がある  

### 検定の手順2,3

検定統計量(分散分析表のF値9を計算する(検定の手順2)  
F値が棄却域に入っているかどうか調べる(検定の手順3)
棄却域に入っている場合、帰無仮設は棄てる

## 使い方

```ruby
xi = [
    [12.2, 18.8, 18.2],
    [22.2, 20.5, 14.6],
    [20.8, 19.5, 26.3],
    [26.4, 32.5, 31.3],
    [24.5, 21.2, 22.4],
]
oneWay = Num4AnovaLib::OneWayLayoutLib.new 
oneWay.oneWay.oneway_anova(xi, 0.05)
=> true
```

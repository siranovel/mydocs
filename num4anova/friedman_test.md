friedman_test
-------------
フリードマン検定

## 統計的検定
### 検定の手順1

・帰無仮設：グループ間に差はない  
・対立仮説：グループ間に差がある  

### 検定の手順2,検定の手順3

検定統計量を計算し、棄却域に入っているかどうか調べる  
棄却域に入っている場合、帰無仮設は棄てる

## 使い方

```ruby
xij = [
    [13.6, 15.6, 9.2],
    [22.3, 23.3, 13.3],
    [26.7, 28.8, 15.0],
    [28.0, 31.2, 15.8],
]
twoWay = Num4AnovaLib::TwoWayLayoutLib.new 
twoWay.friedman_test(xij, 0.05)
=>
  true
```
